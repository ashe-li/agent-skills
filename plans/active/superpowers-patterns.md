# Implementation Plan: Superpowers Patterns Integration

> Generated by manual analysis on 2026-03-26
> Plan file: plans/active/superpowers-patterns.md

## Overview

從 obra/superpowers 擷取 4 個核心模式，整合進現有 ECC 架構。重點在「強制 skill 查詢」機制，解決使用者忘記呼叫 slash command 導致遺漏的問題。

## Implementation Steps

### Phase 1: 強制 Skill 查詢機制（最高優先）

新增 rule `rules/skill-auto-match.md`，讓 Claude 在回應前**主動匹配**現有 skills。

**原理：** Superpowers 用 SessionStart hook 注入 `using-superpowers` skill，強制 agent 每次行動前查詢 skills。我們用 rule 實現（更簡單、不需 hook 腳本），效果等同。

- [ ] Step 1: 建立 `rules/skill-auto-match.md`
  - 定義 task-to-skill 映射表（plan/design 類 → /design、debugging → /systematic-debug、PR → /pr 等）
  - 明確指令：「回應使用者前，先對照映射表檢查是否有匹配的 skill」
  - 「若有 ≥1 個匹配，必須先 invoke Skill tool，再回應」
  - 排除條件：純聊天、簡單問答、已在 skill 執行中
- [ ] Step 2: symlink 到全域 `~/.claude/rules/common/skill-auto-match.md`

### Phase 2: Subagent 隔離模式

新增 rule `rules/subagent-isolation.md`，建立系統性的 subagent 使用原則。

- [ ] Step 3: 建立 `rules/subagent-isolation.md`
  - 原則：預估 >2000 tokens 輸出的 agent task 應使用獨立 subagent
  - 禁止在主 context 中執行大量探索/搜尋；委派 Explore agent
  - subagent 完成後只回傳摘要，不回傳原始輸出
  - 結構化 handoff protocol（沿用 /assist 的 HANDOFF 格式）
- [ ] Step 4: symlink 到全域

### Phase 3: 驗證門檻（Verification Gates）

新增 rule `rules/verification-gates.md`，定義階段性驗證要求。

- [ ] Step 5: 建立 `rules/verification-gates.md`
  - 定義 3 個門檻點：Plan Gate（計畫完成後）、Implementation Gate（實作完成後）、PR Gate（PR 建立前）
  - 每個門檻需要具體證據（test output、build success、lint pass）
  - 「不可聲稱完成但不附證據」
- [ ] Step 6: symlink 到全域

### Phase 4: Systematic Debugging Skill

新增 skill `systematic-debugging/SKILL.md`，取代 PUA 的非結構化激勵。

- [ ] Step 7: 建立 `systematic-debugging/SKILL.md`
  - 4 階段：Observe（收集症狀）→ Hypothesize（列假設）→ Test（驗證假設）→ Fix（修復）
  - 失敗 3 次同一方法後強制升級策略
  - 禁止「盲目重試」和「猜測性修改」
  - 觸發條件：build/test 失敗、error message、debugging context
- [ ] Step 8: 更新 CHANGELOG.md

## Risks & Mitigations

| 風險 | 對策 |
|------|------|
| skill-auto-match 過度觸發，每次回應都要查詢 | 明確定義排除條件；只在「有機會匹配」時觸發 |
| rules 增加 context token 消耗 | 每份 rule 控制在 50 行以內 |
| systematic-debugging 與 PUA 衝突 | 明確分工：debugging 用 systematic-debugging，卡住用 PUA |
